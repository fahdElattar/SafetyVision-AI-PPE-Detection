How to use this app

1. Webcam: Used for real-time monitoring. Click "Use Webcam" to access your local camera hardware.

2. Upload: Drag and drop or select site footage. The file is analyzed and then automatically deleted from the server to protect privacy.

3. Demo: Instant test case. Runs a pre-recorded sequence to demonstrate detection accuracy without setup.
